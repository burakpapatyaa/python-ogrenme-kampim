Python'a aslında yabancı değilim, daha önce de ufak tefek işler yaptım. Ama bir yerden sonra olayın sadece "kopyala-yapıştır" yapmaya veya "ChatGPT'ye yazdırıp geçmeye" dönüştüğünü fark ettim. İleride siber güvenlik, yapay zeka veya web backend tarafında büyük projelere girerken "bu hata nereden çıktı" diye ekrana boş boş bakmamak için bu pas silme kampını başlattım.
